# Year 4 IB Maths Revision Quiz

A fun, interactive two-stage math revision quiz for Year 4 IB students with BLACKPINK-inspired design, cartoon touches, and K-pop vibes! 🎵✨

## Features

### 🎨 Design
- **BLACKPINK Aesthetic**: Dark sleek background with hot pink neon accents
- **Cartoon Touches**: Floating clouds, smooth animations, colorful confetti
- **Responsive**: Works perfectly on desktop, tablet, and mobile

### 🎮 Interactive Features
- **Two Stages**:
  - **Stage 1**: 40 fundamental questions for foundation building
  - **Stage 2**: 40 challenging questions to master the concepts
- **8 Units**: 5-Digit Numbers, Multiples & Factors, Quadrilaterals, Angles, Line Graphs, Time, Decimals, Multiplication
- **Question Types**: Multiple choice and typed answers
- **Instant Feedback**: Detailed explanations for every question
- **Progress Tracking**: Real-time score updates and progress bars
- **Unit Filtering**: Study by topic or all at once

### 🔊 Audio
- **K-pop Inspired Sounds**:
  - Bright ascending chime on correct answers
  - Deep bass thud on wrong answers
  - Victory fanfare when quiz completes
  - Click sounds on all interactions
- **No External Dependencies**: All sounds generated with Web Audio API

### ✨ Visual Feedback
- Confetti bursts on correct answers
- Smooth card animations
- Score animations
- Glow effects and transitions
- Shake animation on incorrect answers

## Files

- **Year4_IB_Maths_Revision.html** - Stage 1: Foundation Level (40 questions)
- **Year4_IB_Maths_Stage2.html** - Stage 2: Challenge Level (40 harder questions)

## How to Use

1. **Open Stage 1**: Open `Year4_IB_Maths_Revision.html` in your web browser
2. **Answer Questions**: Click MCQ options or type answers and press Enter
3. **Check Progress**: Track your score and completion in real-time
4. **Filter by Topic**: Click unit buttons to focus on specific topics
5. **Review Results**: See detailed breakdown of performance by unit
6. **Challenge Yourself**: Move to Stage 2 for harder questions

## Technical Details

- **Language**: HTML5, CSS3, JavaScript (Vanilla)
- **Audio**: Web Audio API (no external libraries)
- **Fonts**: Google Fonts (Outfit, Bebas Neue)
- **No Dependencies**: Completely standalone - no npm, no frameworks
- **Browser Support**: Chrome, Firefox, Safari, Edge (any modern browser)

## Content

### Stage 1 & 2 Coverage

**Unit 1: 5-Digit Numbers** (5 questions each)
- Place values, number ordering, rounding, comparing

**Unit 2: Multiples & Factors** (5 questions each)
- LCM, HCF, common multiples, factorization

**Unit 3: Quadrilaterals** (5 questions each)
- Polygon properties, quadrilateral types, angles

**Unit 4: Angles** (5 questions each)
- Angle types, angles on a line, around a point, in shapes

**Unit 5: Line Graphs** (5 questions each)
- Reading graphs, mean/average, trends, data interpretation

**Unit 6: Time** (5 questions each)
- Time conversions, 12/24-hour clocks, time calculations

**Unit 7: Decimals** (5 questions each)
- Decimal place values, ordering, addition, subtraction

**Unit 8: Multiplication** (5 questions each)
- Mental strategies, grid method, estimation, word problems

## Customization

All content, colors, fonts, and sounds can be easily customized by editing the HTML files:

- **Colors**: Change CSS variables in `:root` selector
- **Questions**: Modify the `questions` array in the `<script>` section
- **Fonts**: Update Google Fonts links in `<head>`
- **Sounds**: Adjust frequency values in audio functions
